# 无人机第一次作业 - 自动飞行控制系统

## 作业内容
基于 AirSim 无人机仿真平台，使用 Python 实现无人机自动起飞、定点巡航、自动降落功能。

## 运行环境
- 操作系统：Windows 10/11 64位
- Python 版本：Python 3.10.11
- 仿真平台：AirSimNH

## 依赖库
- airsim==1.8.1
- numpy>=1.21
- pynput>=1.8
- msgpack-rpc-python>=0.4.1

## 功能实现
1. 自动连接 AirSim 仿真环境
2. 解锁无人机并启动电机
3. 自动起飞至目标高度
4. 按正方形路径自动巡航
5. 自动降落并上锁
6. 任务完成自动退出

## 运行步骤
1. 启动 AirSimNH.exe，选择“否(N)”进入四旋翼无人机模式
2. 等待仿真环境完全加载
3. 打开终端，执行以下命令运行代码：drone_fly.py4. 观察无人机自动完成起飞、巡航、降落全过程

## 项目效果
- 无人机成功连接仿真环境
- 稳定起飞
- 按预定航线飞行
- 精准降落
- 无报错、无异常、可稳定复现
